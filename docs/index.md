# Vibe Code Detector

**Unmask the AI-Generated Web.**

Vibe Code Detector is a free, open-source browser extension that reveals which AI tools, IDEs, and platforms were used to build any website. Instantly detect the "DNA" of Cursor, v0, Lovable, and more — right from your browser.

## Features
- Detects AI IDEs, agents, and platforms
- Heuristic analysis of Tailwind, Shadcn, Radix, Lucide, and more
- Forensic, dark-themed UI
- 100% open source (MIT License)
